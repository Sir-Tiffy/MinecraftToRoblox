This program will convert a minecraft world into a roblox map.

Instructions for use:

1) Launch MinecraftToRoblox.exe
2) Select a minecraft map from the list
3) Press the "Convert" button
4) Select the location for the new roblox place
5) Wait for conversion to complete
6) Open the new place in roblox studio
7) Press the Run button (the green arrow) to build the terrain
8) Wait for terrain to finish building (check the output window for status)
9) Save the game
10) For best results, close roblox and reopen the map
